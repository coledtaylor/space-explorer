# Space Explorer

A web-based space exploration game. Navigate procedurally generated star systems, scan planets and anomalies, and discover the cosmos.

## Play

Open `index.html` in a browser — no build step required.

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Thrust |
| Mouse | Aim ship |
| Click | Scan object |
| E | Interact with nearest object |
| M | Toggle map |

## Features

- Procedurally generated star systems with stars, planets, and anomalies
- Parallax starfield background
- Ship physics with thrust, drag, and fuel management
- Scan panel with detailed object info
- Minimap navigation
- Fuel recharging near stars
- System transitions when exploring beyond boundaries
